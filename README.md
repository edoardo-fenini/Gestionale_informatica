Istruzioni Progetto


Il progetto e costituito da 6 file in totale. Alcuni di essi non hanno un utilizzo diretto, tuttavia per mantenere il progetto aggiornabile è necessario includerli.



Setup PHP

1) avere Xampp installato

2) selezionare tutti i file .php del progeto e aggiungerli in una cartella specificatamente chiamata "gestionale" e aggiungere la cartella nella cartella predefinita di Xampp "htdocs"




Setup SQL

1) aprire php e crare un database con le caratteristiche di quello scritto qua in seguito (forniamo un esempio di sql da copiare "plug and play" (il file con l'esempio è trovabile tra i file nella repository)

2) per sicurezza accertarsi della mancanza di vincoli, potrebbero interferire con il php e non permettere tutte le operazioni in qualsiasi ordine




Setup Python:

1) Consigliamo l'utilizzo di Pycharm per la facilità nell'aggiungere le estenzioni usate nel progetto.

2) Assicurarsi che lo script abbia accesso ai file di Xampp.
